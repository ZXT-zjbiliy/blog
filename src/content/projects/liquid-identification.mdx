## What It Does

LiquidIdentification identifies bottle liquid levels from images. The README describes a YOLO OBB pipeline that can train and predict with three label sets: four-class liquid amount, binary liquid existence, and single-class bottle detection.

## Engineering Shape

The repository keeps the computer-vision work reproducible through scripts for dataset preparation, Roboflow-to-OBB conversion, YOLO training, prediction, and experiment output organization. It also preserves a traditional machine-learning comparison path based on segmented image features, so the project can compare deep-learning detection with interpretable classifiers.

## Why It Is On This Site

This is a compact applied-vision project: it has dataset handling, model training, evaluation-oriented output folders, and practical command-line tooling in one place. It is useful as a record of how a course or research-style computer-vision idea becomes a runnable project.
