# 一、文件操作命令
1. ls
	- 功能：列出目录下的文件/子目录信息
	- 常用选项及含义：
		1. ` -a`：显示所有文件（含隐藏文件，以`.`开头的文件）
		2. ` -l`：以长格式显示（权限、所有者、大小、修改时间等）
		3. ` -h`：配合`-l`使用，以人性化单位（KB/MB/GB）显示文件大小
		4. ` -S`：按文件大小从大到小排序
		5. ` -t`：按修改时间从新到旧排序
		6. ` -r`：反向排序（配合`-S`/`-t`使用）
		7. ` -d`：仅显示目录本身，不列出目录内文件
		8. ` -i`：显示文件的inode编号
2. cd
	- 功能：切换当前终端的工作目录
	- 常用选项及含义：
		1. 无核心选项，通过路径参数实现切换：
			- `cd ..`：返回上一级目录
			- `cd ~`：返回当前用户主目录
			- `cd -`：返回上一次工作目录
			- `cd /绝对路径`：切换到指定绝对路径目录
3. pwd
	- 功能：打印当前工作目录的绝对路径
	- 常用选项及含义：
		1. ` -P`：显示实际物理路径（解析符号链接）
		2. ` -L`：显示逻辑路径（保留符号链接，默认）
4. cat
	- 功能：读取文件全部内容并输出到终端，也可合并文件内容
	- 常用选项及含义：
		1. ` -n`：显示行号（包括空行）
		2. ` -b`：显示行号（仅非空行）
		3. ` -s`：压缩连续空行为单个空行
		4. ` -E`：在每行末尾显示`$`符号
5. more
	- 功能：分页显示文件内容（仅支持向下翻页）
	- 常用选项及含义：
		1. ` -n <数字>`：指定每页显示的行数
		2. ` -d`：显示翻页/退出提示信息
		3. ` -s`：压缩连续空行为单个空行
6. less
	- 功能：灵活分页显示文件内容（支持上下翻页、搜索、跳转）
	- 常用选项及含义：
		1. ` -N`：显示行号
		2. ` -i`：搜索时忽略大小写
		3. ` -S`：截断长行（不换行显示）
		4. ` -e`：文件浏览完毕后自动退出
		5. ` -f`：强制打开特殊文件（如设备文件、目录）
7. touch
	- 功能：创建空文件，或修改文件的访问时间/修改时间
	- 常用选项及含义：
		1. ` -a`：仅修改文件的访问时间
		2. ` -m`：仅修改文件的修改时间
		3. ` -t <YYYYMMDDHHMM[.SS]>`：指定时间戳
		4. ` -c`：若文件不存在则不创建（仅修改已有文件时间）
8. cp
	- 功能：复制文件或目录
	- 常用选项及含义：
		1. ` -r`/`-R`：递归复制目录（含子目录和文件）
		2. ` -p`：保留文件的权限、所有者、时间戳等属性
		3. ` -f`：强制覆盖目标文件（无提示）
		4. ` -i`：覆盖前提示确认（交互式）
		5. ` -u`：仅复制源文件比目标文件新的内容
		6. ` -v`：显示复制过程（详细模式）
9. mv
	- 功能：移动文件/目录，或对文件/目录重命名
	- 常用选项及含义：
		1. ` -f`：强制覆盖目标文件（无提示）
		2. ` -i`：覆盖前提示确认（交互式）
		3. ` -u`：仅移动源文件比目标文件新的内容
		4. ` -v`：显示移动/重命名过程（详细模式）
10. rm
	- 功能：删除文件或目录
	- 常用选项及含义：
		1. ` -f`：强制删除（无提示，忽略不存在的文件）
		2. ` -i`：删除前提示确认（交互式）
		3. ` -r`/`-R`：递归删除目录（含子目录和所有文件）
		4. ` -v`：显示删除过程（详细模式）
11. head
	- 功能：输出文件的前N行内容
	- 常用选项及含义：
		1. ` -n <数字>`：指定显示的行数（默认10行）
		2. ` -c <数字>`：指定显示的字节数
		3. ` -q`：不显示文件名头（多文件时）
		4. ` -v`：显示文件名头（多文件时）
12. tail
	- 功能：输出文件的后N行内容，或实时监控文件变化
	- 常用选项及含义：
		1. ` -n <数字>`：指定显示的行数（默认10行）
		2. ` -c <数字>`：指定显示的字节数
		3. ` -f`：实时跟踪文件新增内容（日志监控常用）
		4. ` -F`：与`-f`类似，文件被删除/重建后仍能跟踪
		5. ` -s <数字>`：配合`-f`使用，指定刷新间隔（秒）
13. sort
	- 功能：对文本文件的行按指定规则排序
	- 常用选项及含义：
		1. ` -n`：按数字大小排序（默认按字符串排序）
		2. ` -r`：反向排序（从大到小/从Z到A）
		3. ` -u`：去重（合并重复行）
		4. ` -k <列数>`：按指定列排序
		5. ` -t <分隔符>`：指定列分隔符
		6. ` -f`：排序时忽略大小写
14. wc
	- 功能：统计文件的行数、单词数、字节数
	- 常用选项及含义：
		1. ` -l`：仅统计行数
		2. ` -w`：仅统计单词数
		3. ` -c`：仅统计字节数
		4. ` -m`：仅统计字符数（区分多字节字符）
		5. ` -L`：显示文件中最长行的字符数
15. cut
	- 功能：从文本每行中提取指定列/字符段
	- 常用选项及含义：
		1. ` -d <分隔符>`：指定字段分隔符（默认制表符）
		2. ` -f <列数>`：指定提取的列（如1、1-3、1,3）
		3. ` -c <字符范围>`：指定提取的字符位置（如1-5、5）
		4. ` -s`：仅显示包含分隔符的行
16. paste
	- 功能：将多个文件的对应行合并为一行
	- 常用选项及含义：
		1. ` -d <分隔符>`：指定合并后的列分隔符（默认制表符）
		2. ` -s`：将单个文件的所有行合并为一行
		3. ` -z`：用NUL字符分隔行（替代换行符）

# 二、搜索命令
1. find
	- 功能：遍历文件系统，按指定条件搜索文件/目录
	- 常用选项及含义：
		1. ` -name <模式>`：按文件名匹配（区分大小写）
		2. ` -iname <模式>`：按文件名匹配（忽略大小写）
		3. ` -type <类型>`：按文件类型筛选（f=文件、d=目录、l=符号链接）
		4. ` -size <大小>`：按文件大小筛选（+n=大于n、-n=小于n，单位：k/M/G）
		5. ` -mtime <天数>`：按修改时间筛选（+n=超过n天、-n=少于n天）
		6. ` -user <用户名>`：按文件所有者筛选
		7. ` -group <组名>`：按文件所属组筛选
		8. ` -perm <权限>`：按文件权限筛选
		9. ` -exec <命令>`：对搜索结果执行指定命令
		10. ` -delete`：删除搜索到的文件/目录
2. grep
	- 功能：在文本中搜索匹配的字符串/正则表达式
	- 常用选项及含义：
		1. ` -i`：忽略大小写
		2. ` -n`：显示匹配行的行号
		3. ` -v`：反向匹配（显示不包含匹配内容的行）
		4. ` -r`/`-R`：递归搜索目录下的所有文件
		5. ` -E`：启用扩展正则表达式（等价于egrep）
		6. ` -F`：将匹配字符串作为固定字符串（不解析正则）
		7. ` -c`：仅统计匹配行的数量
		8. ` -l`：仅显示包含匹配内容的文件名
		9. ` -o`：仅显示匹配的部分（而非整行）
3. locate
	- 功能：基于预生成的文件数据库快速搜索文件名
	- 常用选项及含义：
		1. ` -i`：忽略大小写
		2. ` -c`：仅统计匹配结果数量
		3. ` -l <数字>`：限制显示结果数量
		4. ` -r <正则>`：按正则表达式匹配
		5. ` -u`：更新文件数据库（等价于updatedb）
4. which
	- 功能：查找可执行文件的绝对路径（仅搜索PATH环境变量）
	- 常用选项及含义：
		1. ` -a`：显示所有匹配的路径（默认仅显示第一个）
		2. ` -s`：静默模式，仅返回退出状态（0=找到，1=未找到）
5. ack
	- 功能：程序员专用文本搜索工具，默认忽略版本控制目录（.git/.svn）
	- 常用选项及含义：
		1. ` -i`：忽略大小写
		2. ` -r`：递归搜索（默认开启）
		3. ` -n`：显示匹配行号
		4. ` -l`：仅显示包含匹配内容的文件名
		5. ` -g <模式>`：仅搜索匹配文件名的文件
		6. ` --type <语言>`：仅搜索指定类型文件（如python、java）
6. ag（The Silver Searcher）
	- 功能：高性能代码搜索工具，速度优于ack，默认忽略版本控制目录
	- 常用选项及含义：
		1. ` -i`：忽略大小写
		2. ` -n`：显示匹配行号
		3. ` -l`：仅显示包含匹配内容的文件名
		4. ` -g <模式>`：按文件名筛选
		5. ` -v`：反向匹配
		6. ` -c`：统计匹配次数
7. whereis
	- 功能：查找命令的二进制文件、手册页、源码文件路径
	- 常用选项及含义：
		1. ` -b`：仅查找二进制文件
		2. ` -m`：仅查找手册页
		3. ` -s`：仅查找源码文件
		4. ` -u`：查找不完整的文件（缺少某类文件）
8. type
	- 功能：判断命令类型（内置命令、外部可执行、别名）
	- 常用选项及含义：
		1. ` -t`：仅显示类型（alias=别名、builtin=内置、file=外部文件）
		2. ` -a`：显示所有匹配的命令（含别名、路径）
		3. ` -p`：仅显示外部命令的路径（无则返回空）
9. apropos
	- 功能：通过关键词搜索命令手册的描述信息
	- 常用选项及含义：
		1. ` -d`：显示调试信息
		2. ` -e`：精确匹配关键词
		3. ` -i`：忽略大小写
		4. ` -l`：仅显示手册页名称
		5. ` -w`：按通配符匹配关键词
10. alias
	- 功能：创建/查看命令别名
	- 常用选项及含义：
		1. 无参数：列出所有已定义的别名
		2. ` -p`：以`alias 别名=命令`格式显示所有别名
		3. ` -r`：删除指定别名（等价于unalias）

# 三、目录操作命令
1. mkdir
	- 功能：创建目录
	- 常用选项及含义：
		1. ` -p`：递归创建多级目录（父目录不存在时自动创建）
		2. ` -m <权限>`：创建目录时指定权限（如755）
		3. ` -v`：显示目录创建过程（详细模式）
		4. ` -Z`：设置目录的SELinux安全上下文
2. rmdir
	- 功能：删除空目录
	- 常用选项及含义：
		1. ` -p`：递归删除多级空目录（仅删除空的父目录）
		2. ` -v`：显示删除过程（详细模式）
3. tree
	- 功能：以树形结构显示目录层级及文件
	- 常用选项及含义：
		1. ` -L <层级>`：限制显示的目录层级
		2. ` -h`：以人性化单位显示文件大小
		3. ` -d`：仅显示目录（不显示文件）
		4. ` -i`：不显示树形缩进线
		5. ` -f`：显示文件/目录的完整路径
		6. ` -s`：显示文件/目录的大小（字节）
4. du
	- 功能：统计文件/目录的磁盘占用空间
	- 常用选项及含义：
		1. ` -s`：仅显示总计大小（不显示子目录/文件明细）
		2. ` -h`：以人性化单位显示大小
		3. ` -k`/`-m`/`-G`：分别以KB/MB/GB为单位显示
		4. ` -c`：显示所有文件/目录的总计大小
		5. ` -a`：统计所有文件（默认仅统计目录）
		6. ` -x`：仅统计当前文件系统的文件（不跨挂载点）
5. df
	- 功能：显示文件系统的磁盘空间使用情况
	- 常用选项及含义：
		1. ` -h`：以人性化单位显示大小
		2. ` -i`：显示inode使用情况（而非磁盘空间）
		3. ` -T`：显示文件系统类型（如ext4、xfs）
		4. ` -x <类型>`：排除指定类型的文件系统
		5. ` -P`：使用POSIX标准格式显示

# 四、权限操作命令
1. chmod
	- 功能：修改文件/目录的访问权限
	- 常用选项及含义：
		1. ` -R`：递归修改目录及子目录/文件的权限
		2. ` -v`：显示权限修改过程
		3. ` -c`：仅显示发生变化的修改
		4. ` -f`：忽略错误提示
		5. ` --reference <参考文件>`：复制参考文件的权限
2. chown
	- 功能：修改文件/目录的所有者和所属组
	- 常用选项及含义：
		1. ` -R`：递归修改目录及子目录/文件的属性
		2. ` -v`：显示修改过程
		3. ` -c`：仅显示发生变化的修改
		4. ` -f`：忽略错误提示
		5. ` --reference <参考文件>`：复制参考文件的所有者/组
3. chgrp
	- 功能：修改文件/目录的所属组
	- 常用选项及含义：
		1. ` -R`：递归修改目录及子目录/文件的所属组
		2. ` -v`：显示修改过程
		3. ` -c`：仅显示发生变化的修改
		4. ` -f`：忽略错误提示
		5. ` --reference <参考文件>`：复制参考文件的所属组

# 五、网络操作命令
1. ping
	- 功能：发送ICMP数据包测试网络连通性
	- 常用选项及含义：
		1. ` -c <次数>`：指定发送数据包的次数（默认持续发送）
		2. ` -i <间隔>`：指定数据包发送间隔（秒，默认1秒）
		3. ` -s <大小>`：指定数据包大小（字节）
		4. ` -n`：不解析主机名（仅显示IP）
		5. ` -W <超时>`：指定超时时间（秒）
		6. ` -f`：洪水模式（快速发送数据包，需root）
2. ifconfig
	- 功能：查看/配置网络接口的IP、MAC等参数
	- 常用选项及含义：
		1. 无参数：显示所有网络接口信息
		2. `<接口名>`：显示指定接口信息（如eth0）
		3. ` up`：启用指定接口
		4. ` down`：禁用指定接口
		5. ` netmask <掩码>`：设置接口子网掩码
		6. ` hw ether <MAC>`：修改接口MAC地址（需down后操作）
3. netstat
	- 功能：显示网络连接、路由表、端口监听等状态
	- 常用选项及含义：
		1. ` -t`：仅显示TCP连接
		2. ` -u`：仅显示UDP连接
		3. ` -l`：仅显示监听状态的端口
		4. ` -n`：以数字显示IP/端口（不解析）
		5. ` -p`：显示连接对应的进程ID/名称（需root）
		6. ` -r`：显示路由表
		7. ` -a`：显示所有连接（监听+非监听）
4. ssh
	- 功能：通过SSH协议远程登录主机或执行远程命令
	- 常用选项及含义：
		1. ` -p <端口>`：指定远程主机SSH端口（默认22）
		2. ` -i <密钥文件>`：指定身份验证的私钥文件
		3. ` -X`：启用X11转发（图形界面）
		4. ` -Y`：信任X11转发
		5. ` -q`：静默模式（减少输出）
		6. ` -t`：强制分配伪终端
		7. ` -v`：显示详细连接过程（调试用）
5. scp
	- 功能：基于SSH协议在本地与远程主机间安全复制文件
	- 常用选项及含义：
		1. ` -r`：递归复制目录（含子目录/文件）
		2. ` -P <端口>`：指定远程主机SSH端口（默认22）
		3. ` -i <密钥文件>`：指定身份验证私钥
		4. ` -p`：保留文件的修改时间、权限等属性
		5. ` -q`：静默模式（不显示进度）
		6. ` -C`：启用压缩传输
6. curl
	- 功能：发送HTTP/HTTPS/FTP等请求，获取网络资源
	- 常用选项及含义：
		1. ` -o <文件>`：将响应内容保存到指定文件
		2. ` -O`：按远程文件名保存（URL需指向具体文件）
		3. ` -X <方法>`：指定HTTP请求方法（GET/POST/PUT/DELETE）
		4. ` -d <数据>`：提交POST请求数据
		5. ` -H <头信息>`：添加HTTP请求头
		6. ` -c <cookie文件>`：保存响应的Cookie
		7. ` -b <cookie文件>`：携带Cookie发送请求
		8. ` -L`：跟随重定向
		9. ` -s`：静默模式（不显示进度/错误）
		10. ` -v`：显示详细请求/响应过程
7. telnet
	- 功能：基于TCP协议测试端口连通性或远程登录
	- 常用选项及含义：
		1. ` -l <用户名>`：指定登录的用户名

# 六、进程和系统控制命令
1. ps
	- 功能：显示当前系统的进程状态
	- 常用选项及含义：
		1. ` a`：显示所有用户的进程（含终端进程）
		2. ` u`：显示进程的用户、CPU/内存占用等详细信息
		3. ` x`：显示无终端的进程
		4. ` e`：显示所有进程（等价于`-A`）
		5. ` f`：显示进程树（父子进程关系）
		6. ` -l`：长格式显示
		7. ` -p <PID>`：仅显示指定PID的进程
		8. ` -u <用户>`：仅显示指定用户的进程
2. top
	- 功能：动态实时显示进程的资源占用情况
	- 常用选项及含义：
		1. ` -d <秒>`：指定刷新间隔（默认3秒）
		2. ` -u <用户>`：仅显示指定用户的进程
		3. ` -p <PID>`：仅显示指定PID的进程
		4. ` -b`：批处理模式（输出到文件）
		5. ` -n <次数>`：指定刷新次数后退出
		6. ` -H`：显示线程（而非进程）
3. htop
	- 功能：增强版top，支持鼠标操作、颜色区分
	- 常用选项及含义：
		1. ` -d <秒>`：指定刷新间隔
		2. ` -u <用户>`：仅显示指定用户的进程
		3. ` -p <PID>`：仅显示指定PID的进程
		4. ` -s <列>`：按指定列排序（如CPU、MEM）
		5. ` -t`：显示进程树
		6. ` -C`：禁用颜色显示
4. kill
	- 功能：向进程发送信号（默认终止进程）
	- 常用选项及含义：
		1. ` -l`：列出所有可用信号（如9=SIGKILL、15=SIGTERM）
		2. ` -9`：发送SIGKILL信号（强制终止进程，不可捕获）
		3. ` -15`：发送SIGTERM信号（正常终止，默认）
		4. ` -2`：发送SIGINT信号（等价于Ctrl+C）
		5. ` -1`：发送SIGHUP信号（重启进程）
5. pkill
	- 功能：按进程名/属性匹配并发送信号
	- 常用选项及含义：
		1. ` -i`：忽略进程名大小写
		2. ` -u <用户>`：仅匹配指定用户的进程
		3. ` -signal <信号>`：指定发送的信号（如9、15）
		4. ` -f`：匹配进程的完整命令行（而非仅进程名）
6. shutdown
	- 功能：安全关闭/重启系统
	- 常用选项及含义：
		1. ` -h`：关机（halt）
		2. ` -r`：重启（reboot）
		3. ` -c`：取消已计划的关机/重启
		4. ` -k`：仅发送警告信息，不实际关机
		5. ` now`：立即执行（等价于`+0`）
		6. ` +<分钟>`：指定延迟执行的分钟数
		7. `<HH:MM>`：指定执行的具体时间
7. reboot
	- 功能：重启系统
	- 常用选项及含义：
		1. ` -f`：强制重启（不执行同步操作，慎用）
		2. ` -p`：重启后切断电源（部分硬件支持）
		3. ` -w`：仅记录重启日志，不实际重启
8. logout
	- 功能：退出当前登录会话
	- 常用选项及含义：无核心选项，直接执行即可退出
9. halt
	- 功能：关闭系统（等价于`shutdown -h now`）
	- 常用选项及含义：
		1. ` -f`：强制关机（不执行同步操作）
		2. ` -p`：关机后切断电源
		3. ` -w`：仅记录关机日志，不实际关机

# 七、文本操作命令
1. echo
	- 功能：输出指定文本或环境变量值到终端/文件
	- 常用选项及含义：
		1. ` -n`：不输出末尾的换行符
		2. ` -e`：启用转义字符解析（如\n=换行、\t=制表符）
		3. ` -E`：禁用转义字符解析（默认）
2. printf
	- 功能：格式化输出文本（支持字符串、数字等格式）
	- 常用选项及含义：无核心选项，通过格式符控制输出：
		1. `%s`：字符串
		2. `%d`：整数
		3. `%f`：浮点数
		4. `%c`：单个字符
		5. `\n`：换行
		6. `\t`：制表符
		7. `%-ns`：左对齐，占n个字符
		8. `%.nf`：浮点数保留n位小数
3. sed
	- 功能：流编辑器，对文本行进行替换、删除、新增等操作
	- 常用选项及含义：
		1. ` -i`：直接修改原文件（默认仅输出到终端）
		2. ` -i.bak`：修改原文件并备份为.bak文件
		3. ` -e <脚本>`：指定执行的编辑脚本
		4. ` -f <脚本文件>`：从文件读取编辑脚本
		5. ` -n`：静默模式（仅显示匹配的行）
		6. ` -r`：启用扩展正则表达式
4. awk
	- 功能：按列处理文本，支持条件筛选、计算、格式化输出
	- 常用选项及含义：
		1. ` -F <分隔符>`：指定列分隔符（默认空格/制表符）
		2. ` -v <变量=值>`：定义awk变量
		3. ` -f <脚本文件>`：从文件读取awk脚本
		4. ` -W posix`：启用POSIX兼容模式
5. vi/vim
	- 功能：命令行文本编辑器（vim为vi增强版）
	- 核心模式及操作（无传统选项，按模式区分）：
		1. 命令模式：默认模式，执行复制、删除、跳转等操作
		2. 插入模式：按i/a/o进入，编辑文本内容
		3. 末行模式：按:进入，执行保存、退出、替换等操作
		4. 常用命令模式操作：`yy`（复制行）、`dd`（删除行）、`gg`（跳首行）、`G`（跳末行）
		5. 常用末行模式操作：`w`（保存）、`q`（退出）、`wq`（保存退出）、`q!`（强制退出）、`%s/旧/新/g`（全局替换）
6. nano
	- 功能：简易交互式文本编辑器（新手友好）
	- 常用选项及含义：
		1. ` -w`：禁用行自动换行
		2. ` -l`：显示行号
		3. ` -c`：持续显示光标位置
		4. ` -i`：启用自动缩进
		5. ` -s <程序>`：指定语法高亮程序
7. emacs
	- 功能：全功能文本编辑器/开发环境
	- 常用选项及含义：
		1. ` -nw`：在终端中运行（不启动图形界面）
		2. ` -q`：不加载初始化文件
		3. ` -u <用户>`：加载指定用户的配置
		4. ` -f <函数>`：启动后执行指定函数

# 八、压缩与解压命令
1. tar
	- 功能：创建/解压归档文件，可结合gzip/bzip2压缩
	- 常用选项及含义：
		1. ` -c`：创建新归档文件
		2. ` -x`：解压归档文件
		3. ` -v`：显示归档/解压过程
		4. ` -f <文件名>`：指定归档文件名（必须最后一个选项）
		5. ` -z`：启用gzip压缩/解压（.tar.gz）
		6. ` -j`：启用bzip2压缩/解压（.tar.bz2）
		7. ` -J`：启用xz压缩/解压（.tar.xz）
		8. ` -t`：查看归档文件内容（不解压）
		9. ` -C <目录>`：解压到指定目录
		10. ` --exclude <文件>`：排除指定文件/目录
2. gzip
	- 功能：压缩文件为gzip格式（.gz），或解压.gz文件
	- 常用选项及含义：
		1. ` -d`：解压.gz文件（等价于gunzip）
		2. ` -k`：保留原文件（默认删除原文件）
		3. ` -f`：强制覆盖已存在的压缩/解压文件
		4. ` -l`：显示压缩文件的压缩率、大小等信息
		5. ` -r`：递归压缩目录下的所有文件
		6. ` -1~-9`：指定压缩级别（1最快，9最优，默认6）
3. gunzip
	- 功能：解压gzip格式的.gz文件
	- 常用选项及含义：
		1. ` -k`：保留原压缩文件
		2. ` -f`：强制覆盖已存在的文件
		3. ` -r`：递归解压目录下的.gz文件
4. zip
	- 功能：创建ZIP格式压缩包
	- 常用选项及含义：
		1. ` -r`：递归压缩目录（含子目录/文件）
		2. ` -q`：静默模式（不显示进度）
		3. ` -u`：更新压缩包中的文件（仅添加新/修改的文件）
		4. ` -d`：从压缩包中删除指定文件
		5. ` -m`：压缩后删除原文件
		6. ` -9`：最高压缩级别
		7. ` -1`：最快压缩速度（低压缩率）
5. unzip
	- 功能：解压ZIP格式压缩包
	- 常用选项及含义：
		1. ` -d <目录>`：解压到指定目录
		2. ` -l`：查看压缩包内容（不解压）
		3. ` -q`：静默模式
		4. ` -o`：强制覆盖已存在的文件（无提示）
		5. ` -n`：不覆盖已存在的文件
		6. ` -x <文件>`：排除指定文件/目录

	- 常用选项及含义：
		1. ` -u`：显示当前用户的UID
		2. ` -g`：显示当前用户的主GID
11. date
	- 功能：显示/设置系统日期和时间
	- 常用选项及含义：
		1. ` -s <时间>`：设置系统时间（如"2025-12-06 14:30:00"）
		2. ` -u`：显示UTC时间
		3. ` +<格式>`：自定义输出格式（%Y=年、%m=月、%d=日、%H=时、%M=分、%S=秒）
12. cal
	- 功能：显示日历
	- 常用选项及含义：
		1. ` -1`：显示当前月份（默认）
		2. ` -3`：显示前一个月、当前月、后一个月
		3. ` -y`：显示全年日历
		4. ` -j`：显示日期为一年中的第几天
		5. ` -m`：周一作为一周的第一天（默认周日）

# 十三、其他常用命令
1. man
	- 功能：查看命令/函数的官方手册
	- 常用选项及含义：
		1. ` -f`：等价于whatis，显示命令简短描述
		2. ` -k`：等价于apropos，按关键词搜索手册
		3. ` -w`：显示手册文件的路径
		4. ` -p <程序>`：指定预处理程序
		5. `<章节>`：指定手册章节（1=用户命令、2=系统调用、3=库函数）
2. whatis
	- 功能：显示命令的简短描述（手册摘要）
	- 常用选项及含义：
		1. ` -d`：显示调试信息
		2. ` -v`：显示详细信息
		3. ` -r`：按正则表达式匹配
		4. ` -w`：按通配符匹配
3. history
	- 功能：显示/管理命令执行历史
	- 常用选项及含义：
		1. ` -c`：清空所有命令历史
		2. ` -d <序号>`：删除指定序号的历史命令
		3. ` -a`：将当前会话历史写入历史文件
		4. ` -n`：读取历史文件中未加载的命令
		5. ` -r`：读取历史文件并覆盖当前会话历史
		6. ` -w`：将当前会话历史写入并覆盖历史文件
4. clear
	- 功能：清空终端屏幕
	- 常用选项及含义：无核心选项，快捷键Ctrl+L等价
5. watch
	- 功能：周期性执行指定命令并显示结果
	- 常用选项及含义：
		1. ` -n <秒>`：指定执行间隔（默认2秒）
		2. ` -d`：高亮显示结果变化的部分
		3. ` -t`：不显示标题栏
		4. ` -b`：命令执行失败时发出蜂鸣
		5. ` -e`：命令执行失败时退出
